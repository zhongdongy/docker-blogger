## Eastwind Blogger
A fairly simple static blogging server. All contents cached.

### Example configuration

```yaml
public:
  home_post: welcome
site:
  hostname: "127.0.0.1:5000"
  enable_https: false
  beian:
    enabled: true
    icp_id: 粤ICP备XXX号-1
    beian_id: 粤公安网备XXX号
```
